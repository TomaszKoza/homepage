Week Notes

Week 1

In this week I learned what is front end. Tool needed for this course were presented. I configured my system. Maciek presented the path of Front end developer and topics that we touch and learn during this course. Last the most important lesson was presentation of Website layers.

Week 2

This week Maciek touched basics about HTML and CSS. In HTML I learned about basic TAGs and general structure of HTML. Regarding CSS I know how to attach CSS style sheet to HTML how to implicate style in HTML document booth in HEAD or BODY part. Now thanks CSS I can change view of page faster and more efficient. We used validator to exanimate our code. We learned about devtool in google chrome and how it use and change look of website. At the end Maciek showed us useful tool to plan our tasks- KANBAN and recommend trello page to use it.

Week 3

In this week we I think most important thing were tools that are used for front end. I learned what is GIT and GITHUB also what are differences between them. Quite usful thing was naming of classes in CSS there are 4 popular styles: kebab-style, snake_style, CamelStyle and pascalStyle. Thanks to lesson about HTML protocol I understand better of communication and information exchange between webbrowser and server. There was lesson about basic of terminal. In case of CSS I learned about classes and styles heritage. In case of HTML topic of atributes were touched. Finally we started learn JavaScript, actually it's basics but really important: how to attach JavaScript to HTML how to operate in console in DevTool and basic about JS.

Week 4

This week was full of extremaly useful informations and tools. First of all I know how to use live server. In case of classes I learned what is BEM (Block Element Modifier) and how to use this concept to name classes in CSS. There were many information about positioning, displaying and owerflowing. Also box-model concept was introduced. Next portion of information regarding JavaScript allow us to pull parts of HTML to the console and how to replace it and change.
